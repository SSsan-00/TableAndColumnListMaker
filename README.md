# TableAndColumnListMaker

Excel VBA で、指定フォルダ配下のブックを再帰的に探索し、対象ブックの先頭シートからテーブル・カラム情報を抽出するための解析マクロです。

## 現在決まっていること

- ユーザーが選択したフォルダを起点に、サブフォルダを含めて再帰的に探索します。
- ファイル名が次の形式に一致する Excel ブックを解析対象にします。
  - `1`
  - `2`
  - `3`
  - `4`
  - `5`
  - `1-000` から `5-999`
- 対象ブックの先頭シートを解析します。
- ソースは GitHub で公開管理します。
- ソース一式をダウンロードできないユーザー向けに、単一ファイルで導入を開始できる Bootstrap 方式を用意します。
- 導入手順は日本語で整備します。

## これから一緒に決めること

- 対象拡張子を `.xlsx` / `.xlsm` / `.xls` / `.xlsb` のどこまで含めるか。
- ファイル名判定で拡張子以外の前後文字を許可するか。
- 枝番付き形式を `1-001` のような3桁固定にするか、`1-1` / `1-01` / `1-001` を許可するか。
- 先頭シートからどのセル範囲をテーブル・カラム情報として読むか。
- 出力先を新規ブック、実行ブック内シート、CSV のどれにするか。
- エラー時に処理を止めるか、スキップしてログへ残すか。
- Bootstrap ファイルの形式を `.bas`、`.vbs`、`.xlsm` のどれにするか。

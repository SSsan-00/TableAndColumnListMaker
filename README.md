# TableAndColumnListMaker

Excel VBA で、指定フォルダ配下のブックを再帰的に探索し、対象ブックの先頭シートからテーブル・カラム情報を抽出するための解析マクロです。

## 現在決まっていること

- ユーザーが選択したフォルダを起点に、サブフォルダを含めて再帰的に探索します。
- ファイル名に次の形式を含む Excel ブックを解析対象にします。
  - `1`
  - `2`
  - `3`
  - `4`
  - `5`
  - `1-` から `5-` の後ろに数字1文字から3文字が続く形式
  - `7-` の後ろに数字1文字から3文字が続く形式
- 対象ブックの先頭シートを解析します。
- ハイフン後の枝番は `1` / `01` / `001` のような最大3桁を許可します。
- 対象拡張子は `.xlsx`、`.xlsm`、`.xlsb`、`.xls` です。
- 解析結果として `テーブル一覧`、`カラム一覧`、`テーブル検索`、`カラム検索` の4シートを作成します。
- テーブル名（和）とテーブルIDは先頭シート名から分解し、日本語を含む塊はテーブル名（和）、日本語を含まない英数字の塊はテーブルID候補として扱います。
- 解析マクロ本体は [src/TableAndColumnListMaker.bas](src/TableAndColumnListMaker.bas) です。
- 単一ファイル Bootstrap は [bootstrap/InstallTableAndColumnListMaker.vbs](bootstrap/InstallTableAndColumnListMaker.vbs) です。
- ソースは GitHub で公開管理します。
- ソース一式をダウンロードできないユーザー向けに、単一ファイルで導入を開始できる Bootstrap 方式を用意します。
- 導入手順は日本語で整備します。

## これから一緒に決めること

- パスワード付きブック、破損ブック、長いパスなどをどこまで詳細にログ化するか。
- 検索シートの数式行数を初期値の1000行から変更できるようにするか。

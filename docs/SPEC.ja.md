# 仕様メモ

## 目的

選択したフォルダ配下にある対象 Excel ブックを再帰的に探索し、各ブックの先頭シートからテーブル名・カラム名などの一覧を作成する。

## 探索仕様

1. ユーザーがフォルダ選択ダイアログで起点フォルダを選ぶ。
2. 起点フォルダ配下のファイルを再帰的に探索する。
3. Excel の一時ファイル `~$` で始まるファイルは除外する。
4. ファイル名の拡張子を除いたベース名が次のいずれかに完全一致する場合、解析対象にする。
   - `1`
   - `2`
   - `3`
   - `4`
   - `5`
   - `7-000` から `7-999`

## ファイル名判定案

正規表現で表す場合は次の案を初期候補とする。

```text
^(?:[1-5]|7-\d{1,3})$
```

ただし、`7-000` のようなゼロ埋めあり最大3桁という表現を厳密に扱うかは要確認。

- 案A: `7-0`、`7-00`、`7-000`、`7-999` を許可する。
- 案B: `7-000` から `7-999` の3桁だけを許可する。
- 案C: `7-1` と `7-001` を同じ番号として扱う。

## 解析仕様

- 対象ブックは読み取り専用で開く。
- 解析対象は先頭シートのみとする。
- 具体的な読み取り範囲、ヘッダー行、テーブル名・カラム名の判定規則は未決定。

## 出力仕様

未決定。候補は次の通り。

- 実行ブック内に一覧シートを作成する。
- 新しいブックを作成して一覧を出力する。
- CSV または TSV として保存する。

## Bootstrap 方針

ソース一式をダウンロードできないユーザー向けに、単一ファイルから導入を開始できる仕組みを用意する。

候補:

- `.bas` 形式の単一 VBA モジュールを配布し、Excel の VBA エディタからインポートして実行する。
- `.vbs` 形式の単一スクリプトで GitHub 上の VBA ソースを取得し、Excel ブックへ組み込む。
- 配布用 `.xlsm` を生成し、マクロ有効ブックとしてそのまま実行する。

## 未決定事項

- 対象拡張子。
- パスが長い場合の扱い。
- パスワード付きブックの扱い。
- 非表示シート、保護シートの扱い。
- 解析結果に含める列。
- 処理ログの形式。
- 既存出力シートがある場合の上書き確認。
